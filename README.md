🔥❄️ AR Fire & Ice Magic System

An interactive Augmented Reality (AR) Hand Tracking Project built with Python, OpenCV, and MediaPipe. The system tracks both hands in real time and creates stunning Fire and Ice visual effects using glowing particles, crystal trails, energy orbs, and motion-based bursts.

✨ Features
🖐️ Real-time hand detection and tracking
👆 Index finger tracking with smooth motion
🔥 Right Hand → Fire Magic Effects
❄️ Left Hand → Ice Magic Effects
💎 Crystal-style particle system
✨ Twinkling and shimmering particle effects
🌊 Motion trails following finger movement
🎇 Burst effects on fast hand movements
🌟 Glowing energy orbs on palms
🖥️ Fullscreen AR experience
⚡ Smooth particle physics and gravity effects
🛠️ Technologies Used
Python
OpenCV
MediaPipe
NumPy
